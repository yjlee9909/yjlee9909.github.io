## 기본 선택자

### 전체 선택자
*를 사용
주로 웹 브라우저 기본 스타일 초기화 할 때 사용한다.

- 기본 코드

~~~
<style>
    * {
      background-color: black;
      color: aliceblue;
    }
</style>
~~~

### 타입 선택자
특정 태그를 사용하는 모든 요소에 스타일 적용

- \<head> 코드

~~~
p {
      background-color: black;
      color: aliceblue;
}
~~~

- \<body> 코드

~~~
<body>
  <h1>hi</h1>
  <p>안녕하세요?</p>
  <p>ddddd</p>
</body>
~~~
